# retries-queue

[![npm version](https://img.shields.io/npm/v/retries-queue.svg)](https://www.npmjs.com/package/retries-queue)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Promise utilities — allSettled polyfill, race with fallback, sequential execution

## Installation

```bash
npm install retries-queue
```

## Functions

- `allSettledResults` — Promise.allSettled with typed results
- `raceWithFallback` — Promise.race with a fallback if all fail
- `sequential` — Execute async functions sequentially (not in parallel)
